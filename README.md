## Bezaha — site web (vitrine)

Version web inspirée de l’application **Bezaha** (guide touristique à Madagascar).

### Démarrage (local)
- Ouvrir `index.html` dans un navigateur.

### Structure
- `index.html` : page principale
- `styles.css` : styles (thème sombre + carrousels + modals)
- `app.js` : données + navigation + carrousels + Google Maps (iframe)
- `img/` : images utilisées par le site
